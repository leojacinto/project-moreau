# project-moreau

Consolidated repository of ServiceNow-focused projects. Intended as a training corpus and reference for AI coding agents (Claude Code, Windsurf Cascade, ServiceNow Build Agent, etc.).

## Modules

| Directory | What it is |
|-----------|-----------|
| `project-virgil` | Presales intelligence platform - architecture chat, pricing engine, solution recommendations |
| `project-virgil-ba` | Virgil business architecture variant |
| `project-sybil` | Travel Request app benchmarking AI-assisted scoped app development on ServiceNow |
| `project-basanos` | Reference implementation - AI agents discovering and enforcing rules across enterprise systems |
| `project-khepri` | Experimental app proving AI agents can programmatically create ServiceNow AI Agents via Now SDK Fluent DSL |
| `khepri-travel-agent` | Corporate travel policy agent built with Now SDK Fluent API |
| `project-ariadne` | (Placeholder - empty) |
| `AILearningDay` | AI Learning Day agenda app with public API endpoints |
| `AILearningDay-Fluent` | AI Learning Day agenda builder - Fluent API variant |
| `FedForumDemo` | Federal cleared starter onboarding demo |
| `DemoHubPublicHardening` | (Placeholder - empty) |
| `student_management` | Student management scoped app |

## Usage

Point your agent at the root or any subdirectory. Each module is self-contained with its own README where available.
